# Colour-Palette-Gen
A colour palette generator that I made to help me with my graphical design sense given I struggle with picking colours that go together.

Wasn't really necessary given services that provide the same functionality exist, e.g. [Coolers](https://coolors.co), but it was a fun, and I had a small amount of time to spare.

Colour Schemes (Can be seen implemented here [colour-generation.js](/src/utility/colour-generation.js)):
* Monochromatic
* Analogous
* Complementary
* Split Complementary
* Triadic
* Tetradic

> More: [Wiki Article on Colour Schemes](https://en.wikipedia.org/wiki/Color_scheme)
